# Writerside Onboarding Guide

This repository contains the source files for a "Tool Onboarding" guide created as part of the JetBrains Technical Writer recruitment process.

The project demonstrates how to set up a local development environment using **IntelliJ IDEA** and **Writerside** to author and publish technical documentation.

## Project Overview

The documentation covers the following workflow:
* Installing and configuring the Writerside plugin.
* Creating a new documentation project from a Starter template.
* Creating and populating a topic (for the specific purpose of the recruitment test).
* Building and verifying a Web Archive.
* Sharing the project to a public GitHub repository.

## Tools Used

* **IDE:** [IntelliJ IDEA](https://www.jetbrains.com/idea/)
* **Documentation Engine:** [Writerside](https://www.jetbrains.com/writerside/)
* **Version Control:** Git / GitHub

## How to View the Source

The primary documentation content is located in the `Writerside/topics` directory. 
* **Main Topic:** `onboarding-guide.md` (or your specific filename)

To open this project in Writerside:
1. Clone this repository.
2. Open IntelliJ IDEA and select **File | Open**.
3. Navigate to the cloned folder and select the `Writerside` directory.
